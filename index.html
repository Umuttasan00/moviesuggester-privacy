Privacy Policy

Effective: 2025-09-10

1. Introduction

This page explains the privacy practices of the Movie Suggester mobile application. The app provides movie information and does not require user registration, payment, or account creation.

2. Data We Collect

  • The app does not collect personally identifiable user data.
  • Favorites and Watchlist are stored only on the device (Core Data).
  • Users can save movie covers to the device's photo album; this data is only stored locally.
  • Search terms are stored on the device only and are not transmitted to any server.

3. Third-Party Services

The app may fetch movie posters and images from the internet to display content. These requests do not include any personal user data and are not stored by the app.

4. Data Security

The app does not send user data to external servers. Data stored on the device (favorites, Watchlist, saved covers) is protected using standard iOS security measures.

5. Changes

This privacy policy may be updated from time to time. The “Effective” date at the top of this page will be updated to reflect important changes.
